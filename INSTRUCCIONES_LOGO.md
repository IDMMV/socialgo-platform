# Logo MiZona

Copia la carpeta `assets` dentro de la raíz de tu repositorio y acepta reemplazar los archivos existentes.

Archivos principales:
- `mizona-logo.svg`: logo vertical/cuadrado.
- `mizona-logo-horizontal.svg`: logo para fondos claros.
- `mizona-logo-blanco.svg`: logo para barra lateral oscura.
- `mizona-icon-app.svg`: icono maestro de la aplicación.
- `icon-192.png` y `icon-512.png`: iconos para manifest/PWA.
- `apple-touch-icon.png`: icono para iPhone/iPad.
- `favicon-32.png`: favicon.

Nota: las páginas actuales que muestran `MZ` mediante HTML/CSS no cambiarán solo por copiar las imágenes; también deben usar estas rutas en el encabezado o la barra lateral.
