# SOCIALGO — FASE 8.1: REDISEÑO VISUAL

Diseño aplicado: Concepto B.

## Cambios principales

- Interfaz oscura más moderna.
- Menú lateral limpio.
- Barra de búsqueda superior.
- Historias dentro de una tarjeta.
- Feed central más destacado.
- Panel derecho con solicitudes, amigos y tendencias.
- Barra inferior optimizada para celular.
- Botón administrativo visible solo para administradores.
- Aviso de Premium como función futura, sin activación ni cobro.
- Mantiene las funciones de las fases anteriores.

## Instalación con GitHub Desktop

1. Descomprime este ZIP.
2. En GitHub Desktop abre `socialgo-platform`.
3. Pulsa `Show in Explorer`.
4. Copia todo el contenido descomprimido.
5. Selecciona `Reemplazar los archivos en el destino`.
6. En Summary escribe: `Aplicar rediseño SocialGo Concepto B`.
7. Pulsa `Commit to main`.
8. Pulsa `Push origin`.

## Supabase

No necesitas ejecutar SQL. Esta actualización es solamente visual.
