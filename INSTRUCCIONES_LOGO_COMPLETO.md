# MiZona.pe — proyecto completo con logo oficial

1. Descomprime el ZIP.
2. Copia todo el contenido de la carpeta `MiZona_Completo_Logo_Actualizado` dentro de tu repositorio.
3. Acepta reemplazar los archivos existentes.
4. En GitHub Desktop usa el comentario: `Actualizar logo oficial en toda MiZona`.
5. Presiona **Commit to main** y luego **Push origin**.
6. Abre `https://mizona.pe` y realiza una recarga completa.

## En celulares donde la PWA ya estaba instalada

El icono puede permanecer en caché. Desinstala el acceso de la pantalla de inicio y vuelve a instalar MiZona desde `https://mizona.pe`.

## Archivos integrados

- Logo blanco para barras laterales oscuras.
- Logo horizontal para acceso, registro y cabeceras claras.
- Iconos de 192 y 512 píxeles para la PWA.
- Apple Touch Icon y favicon.
- Nueva versión de caché del service worker.
